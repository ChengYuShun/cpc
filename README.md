## 标准
### 基本标准
* 推荐使用驼峰命名发命名
* 源文件使用 **utf-8** 编码
* 所有关键字均为大写，并且程序有大小写区分
* 程序使用大量保留字以实现子空间~~（虽然官方文档要求有缩进，但是在此不做强制要求）~~
* 使用 `//` 进行注释

### 数据类型
* **INTEGER** 整型
* **REAL** 浮点数
* **CHAR** 单个字符
* **STRING** 字符串
* **BOOLEAN** 布尔值
* **DATE** 任何合法的日期 (此类型暂时不会实现)
